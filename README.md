# 🚀 Selenium Automation - Timesheet Management System

## 🔧 Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- Page Object Model (POM)  
- Data Driven Testing (Excel)  

---

## 📌 Features
- Automation of Timesheet Management workflows  
- Validation of timesheet submission, approval, and rejection processes  
- Multi-level approval workflow testing  
- Partial and bulk approval/rejection scenarios  
- Data-driven testing using Excel  
- Reusable and maintainable Hybrid Framework design  
- Test execution using TestNG  
- Generates detailed reports using Extent Reports  

---

## ▶️ How to Run
1. Clone the repository  
2. Import as a Maven project in IntelliJ IDEA  
3. Update test data in Excel file (if required)  
4. Run `testng.xml`  
5. View execution reports inside the `/reports` folder  

---

## 📂 Project Structure
- pageObjects/  
- testClasses/  
- utilities/  
- testData/  
- reports/  
- logs/  

---

## 🧪 Scenarios Covered
- Timesheet time logging  
- Timesheet submission for single and multiple projects  
- Multi-level approval workflows  
- Manual and auto approval scenarios  
- Rejection and partial rejection scenarios  
- Bulk approval and rejection  
- Role-based access validation
